📋 프로젝트 개요

본 프로젝트는 동서대학교에서 사용할 좌석 배치도 관리 및 기자재 대여 시스템입니다. WPF(Windows Presentation Foundation)를 기반으로 한 데스크톱 애플리케이션이며, 학생 정보 관리, 좌석 배치, 기자재 대여 현황, 승인 관리 등의 기능을 제공합니다.

🎨 UI/UX 개선사항 완료

✅ 삼선 메뉴 아이콘 제거

왼쪽 상단 대시보드 헤더의 불필요한 삼선 아이콘을 제거하고 깔끔한 UI로 정리

✅ 메모 패널 확대

메모 기능 패널의 가로 해상도를 100px 증가 (220px → 320px)하여 더 많은 콘텐츠 표시 가능

✅ 메모 수정 버튼 추가

각 메모 항목에 삭제 버튼 옆에 수정 버튼(✎)을 추가하여 메모 편집 기능 제공

✅ 모달 수정 버튼

좌석 클릭 시 나타나는 상세정보 모달에 녹색 '수정' 버튼 추가

✅ 알림 시스템 개선

상단 알림 뱃지를 3개로 분리: 상상Lab 승인 대기, 캐비닛 승인 대기, 기자재 반납

✅ 설정 탭 URL 입력

구글 폼 URL을 설정 탭에서 입력하고 수정할 수 있는 텍스트박스 추가

⚙️ 기능 구현 현황

기능 상태 설명
좌석 고정 토글 ✅ 완료 고정된 좌석을 다시 선택하면 고정 상태가 해제되도록 구현
새로고침 버튼 ✅ 완료 데이터 관리에서 학생 정보를 새로고침하여 대시보드에 갱신

🎯 동작 원리

Step 1: 초기 빌드

auto-build.bat를 실행하면 먼저 전체 프로젝트를 빌드합니다.

Step 2: 파일 감시

*.xaml, *.xaml.cs, *.cs 파일의 변경을 실시간으로 감시합니다.

Step 3: 자동 빌드

파일이 변경되면 자동으로 dotnet build를 실행합니다.

Step 4: 자동 실행

빌드 성공 시 기존 앱을 종료하고 새 버전을 자동으로 실행합니다.

💡 사용 팁

  • 코드 에디터와 auto-build 창을 나란히 띄워서 작업하면 효율적입니다.
  • 빌드 오류는 콘솔 창에서 실시간으로 확인할 수 있습니다.
  • Ctrl+C를 눌러 모니터링을 중단할 수 있습니다.
  • 종료 시 창을 닫기만 해도 자동으로 종료됩니다.
